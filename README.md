# CS ASPNET Core Multitenancy

Project sample with CS ASPNET Core for Multitenancy

## Setup:

Create/Update DB migration

```bash
update-database
```

## Migrations

Add new migration

```bash
add-migration "example name migration"
update-database
```

Delete database

```bash
drop-database
```
